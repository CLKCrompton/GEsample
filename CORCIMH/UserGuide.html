User Guide


HTML CRITICAL EDITION

  • This edition is a facsimile mark-up using HTML. The pages of the site are linked, each representing a page of the manuscript. Each page contains a picture of the page in the codex that it represents, and can be engaged with. Beside the picture is an HTML mark-up of the original text, including notes on the manuscript, as well as textual decisions that represent important aspects of the manuscript. These decisions are outlined below.

  • Italics represent letters that are different somehow in the manuscript. Either in the form of large, illuminated letters or redacted text.

  • Note that line endings are respected.

  • Superscript is used to denote letters that have been redacted in favour of a superscript apostrophe-like remark.

  • Blue text denotes titles.

  • Red Letters are Illuminated initials or text that is coloured to match the manuscript.

  • Pages are navigated by linked text in top right corner of the page

  • The hyphen indicates the word continues on the next line.


Legend

  • ¶ = Pilcrow marks the beginning of a new paragraph

  • [fol. 1r]= recto folio [each page on the right in a codex; in this case, number one]

  • [fol. 1v]= verso folio [each page on the left in a codex; in this case, number one]

  • [col. A]= denotes start of column [each page has two columns, in this case this is the first column, A]
